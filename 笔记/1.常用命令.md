# 使用yarn
## 一、安装yarn 
- **使用Corepack管理安装yran，入过使用 npm install -g yarn 安装，可能出现权限或PATH问题**
1. 以管理员方式打开命令行输入：
   ```corepack enable```
2. 然后在运行：
   ```corepack prepare yarn@stable --activate```
3. 验证是否安装成功
   ```yarn --version```

## 二、初始化项目

1. yarn初始化项目
   ```yarn init -y```
2. 添加化hardhat
   ```yarn add -D hardhat```
3. 初始化hardhat
   ```npx hardhat```


# 使用npm
## 自定义创建项目
1. 初始化项目
   ```npm init -y```
2. 安装hardhat（作为开发依赖），如果不加@版本号，默认安装最新版
   ```npm install --save-dev hardhat@2.22.10```
3. 初始化hardhat项目
   ```npm hardhat```

## hardhat常用命令

1. 查看项目hardhat版本
   ```npx hardhat --version```
2. 编译
   ```npx hardhat compile```
3. 测试
   ```npx hardhat test```
4. 部署
   ```npx hardhat run ./scripts/deploy_HelloWorld.ts --network localhost```
5. 启动本地网络
   ```npx hardhat node```

## 添加Ts库
```npm install --save-dev typescript ts-node @types/node```
